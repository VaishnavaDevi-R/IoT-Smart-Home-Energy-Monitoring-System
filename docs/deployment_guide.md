# Deployment Guide

## Step 1: Push Code to GitHub

```bash
git add .
git commit -m "Final project version"
git push origin main
```

---

## Step 2: Verify Repository Structure

Ensure the repository contains:

* Source Code
* Documentation
* Images
* Reports
* Screenshots
* README

---

## Step 3: Deploy Dashboard

Visit:

https://share.streamlit.io

Login with GitHub.

---

## Step 4: Select Repository

Repository:

IoT-Smart-Home-Energy-Monitoring-System

Branch:

main

Main File:

dashboard/streamlit_dashboard.py

---

## Step 5: Deploy

Click Deploy.

The dashboard will be available through a public URL.

---

A fully accessible cloud-hosted Smart Home Energy Monitoring Dashboard.
