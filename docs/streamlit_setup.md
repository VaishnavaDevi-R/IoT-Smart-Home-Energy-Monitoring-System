# Streamlit Dashboard Setup

## Install Streamlit

```bash
pip install streamlit
```

---

## Run Dashboard

```bash
streamlit run dashboard/streamlit_dashboard.py
```

---

## Access Dashboard

Open:

```text
http://localhost:8501
```

---

## Dashboard Features

### Live Metrics

* Voltage
* Current
* Power
* Cost

### Charts

* Power Trend
* Energy Trend
* Cost Analysis
* Daily Usage
* Monthly Usage

### Data Table

Displays recent energy records.
